# Concepto de Diseño: Catálogo de Comidas Rápidas y Viandas

## Enfoque Seleccionado: "Apetito Moderno"

### Filosofía de Diseño
Un catálogo de comidas que combina **calidez artesanal** con **modernidad limpia**. El objetivo es hacer que cada producto se sienta apetitoso y accesible, con una navegación intuitiva que invite a explorar y ordenar.

### Principios Fundamentales
1. **Apetencia Visual:** Las imágenes de productos son protagonistas. Cada foto debe verse fresca, apetitosa y de alta calidad.
2. **Jerarquía Clara:** Categorías bien diferenciadas (Comidas Rápidas, Viandas Saladas, Viandas Dulces) con navegación fluida.
3. **Confianza y Accesibilidad:** Interfaz intuitiva que transmita profesionalismo sin ser intimidante.
4. **Velocidad Percibida:** Transiciones suaves y carga rápida para que el usuario se sienta en control.

### Filosofía de Colores
- **Primario:** Naranja cálido (`#FF6B35`) - transmite energía, apetito y calidez
- **Secundario:** Verde fresco (`#2D6A4F`) - representa frescura, salud, ingredientes naturales
- **Neutro:** Blanco limpio (`#FFFFFF`) y gris suave (`#F5F5F5`) - para respiro visual
- **Acentos:** Rojo suave (`#D62828`) para destacar ofertas o elementos interactivos

**Intención Emocional:** Que el usuario sienta hambre y confianza al mismo tiempo.

### Paradigma de Layout
- **Hero Section:** Imagen grande y atractiva del negocio o un plato destacado
- **Navegación de Categorías:** Tabs o cards horizontales que permitan cambiar entre categorías sin scroll excesivo
- **Grid de Productos:** Tarjetas con imagen, nombre, descripción breve y botón de WhatsApp
- **Footer Minimalista:** Información de contacto y horarios

### Elementos Distintivos
1. **Tarjetas de Productos con Sombra Suave:** Efecto de profundidad sin ser excesivo
2. **Badges de Categoría:** Etiquetas visuales que identifiquen rápidamente el tipo de producto
3. **Animación al Hover:** Las tarjetas se elevan ligeramente y la imagen se amplía sutilmente

### Filosofía de Interacción
- **Botones de WhatsApp Prominentes:** Verde WhatsApp (`#25D366`) que invita a la acción
- **Feedback Inmediato:** Confirmación visual cuando el usuario interactúa
- **Scroll Suave:** Transiciones fluidas entre secciones

### Animaciones
- **Entrada de Tarjetas:** Fade-in con desplazamiento suave (100ms)
- **Hover en Productos:** Elevación sutil (transform: translateY(-4px)) con sombra aumentada (200ms)
- **Botones:** Escala 0.97 al presionar para confirmar la interacción
- **Transiciones de Categoría:** Fade entre categorías (150ms)

### Sistema Tipográfico
- **Títulos (Display):** Fuente sans-serif bold (ej: Poppins Bold) - impactante y moderna
- **Subtítulos:** Fuente sans-serif medium (ej: Poppins Medium) - clara y legible
- **Body Text:** Fuente sans-serif regular (ej: Poppins Regular) - cómoda de leer
- **Jerarquía:** Títulos grandes (32-40px), subtítulos (18-24px), body (14-16px)

---

## Resumen Visual
- **Atmósfera:** Apetitosa, moderna, accesible
- **Paleta:** Naranja, verde, blanco, con acentos rojos
- **Tipografía:** Poppins (bold, medium, regular)
- **Animaciones:** Suaves, rápidas, intuitivas
- **Objetivo:** Que el usuario sienta hambre y compre con confianza
